🚀 Digi Tools Hub

A modern web application that helps users build resumes, track job applications, and prepare for interviews — all in one place.

📖 Description

Digi Tools Hub is a powerful and user-friendly platform designed for job seekers. It provides essential tools like resume building, job tracking, and interview preparation to make the job search process easier and more efficient.

🛠️ Technologies Used
⚛️ React.js
🎨 Tailwind CSS + DaisyUI
📦 JavaScript (ES6+)
🌐 Fetch API
⚡ Vite


✨ Features
🔹 1. Smart Resume Builder

Create professional resumes with modern templates and export them easily as PDF.

🔹 2. Job Application Tracker

Track all your job applications, interviews, and rejections in one dashboard.

🔹 3. Interactive UI with Dynamic Data

Fetch and display product/tools data dynamically with smooth user experience.
